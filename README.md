## Folder Overview

```shell
.
├── hardware # Hardware resources: schematics, BOM, gerber, and component placement files
├── other # User guides, module datasheets, and other documentation
└── software # Software resources
```

1. hardware

The hardware is designed using Lichuang EDA, and the full project will be open-sourced later. Currently, schematics, Gerber files, and other resources are available.

2. other

This folder mainly provides module datasheets, 3D models, CAD dimension drawings, pin definitions, label files, and other resources.

3. software

This folder mainly includes the ESP-IDF blink example, MicroPython firmware, AT firmware, and other software resources.

The environment setup tutorials are available on the online wiki.
