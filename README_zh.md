## 文件夹预览

```shell
.
├── hardware # 硬件资料，主要存放原理图，bom，gerber，位号图
├── other # 使用指南，模组规格书等其它资料
└── software # 软件资料
```

### hardware

硬件使用立创EDA制作，后续会工程开源。目前开放原理图、gerber等资料

### other

主要开放模组规格书、3D模型、CAD尺寸图、引脚定义、贴纸文件等资料

### software

主要开放ESP-IDF的blink例程。micropython固件、AT固件等资料

其中搭建环境教程有在在线wiki上有教程。
